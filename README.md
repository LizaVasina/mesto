**О проекте**
* Cервис Mesto: интерактивная страница, куда можно будет добавлять фотографии, удалять их и ставить лайки и редактировать данные профиля

**Технологии**
* HTML5, CSS3, именование классов и файловая структура в соответствии с методологией БЭМ, адаптивная верстка, JavaScript
